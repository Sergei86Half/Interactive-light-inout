Создайте файл:
Нажмите Win + R, введите notepad и нажмите Enter.
Скопируйте код из файла light inout.go в Блокнот.
Нажмите Файл -> Сохранить как....
Назовите файл light.go (убедитесь, что тип файла — "Все файлы").

Подготовьте среду:
Убедитесь, что у вас установлен Go.
Откройте cmd (Win + R -> cmd -> Enter).
Установите необходимый модуль командой:

go get golang.org/x/term

Запуск:
В командной строке введите:

go run light.go
